Projects

Blooming Hotel
  • HTML
  • CSS

A mock website that was developed with vanilla HTML and CSS. The website design was inspired by the The Empire Hotel's website. In this project, I learnt how to use flexbox, create mobile responsive design, and optimise images for performance.

This project was completed as part of an online course, Modern HTML & CSS From The Beginning (Including Sass).

Advice Generator App
  • HTML
  • Tailwind CSS
  • JavaScript

A web application that generates a new piece of advice by clicking the dice icon. I learnt how to fetch data from a public API and access the JSON object to get a string of advice and its ID. While data is fetching, I've added a loader and disabled the dice button.

This project was completed as part of the Advice generator app challenge on Frontend Mentor.

Notifications Page
  • Tailwind CSS
  • React
  • Next.js

A notifications page that allows users to mark unread notifications as read. When selecting 'mark all as read', it removes the active state of unread notifications and resets the total number of unread notifications to zero.

This project was completed as part of the Notifications page challenge on Frontend Mentor.
